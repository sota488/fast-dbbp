# Open Questions

このファイルは、現時点の設計で確定した仕様を記録し、残る未確定事項を明確にするためのものです。

## 1. 確定した正式仕様
- 全ハンド Bomb Potで開始する。
- Small Blind (SB) は存在しない。
- Big Blind (BB) は存在しない。
- すべての参加プレイヤーは毎ハンド必ず2BBを強制投入する。
- 開始Potは `参加人数 × 2BB` で計算する。
- Board A / Board B の2つの独立ボードを使用する。
- ゲームはプリフロップなしで Flop から開始する。
- Turn を経由する。
- River を経由する。
- Showdown で勝者を判定する。
- Foldしたプレイヤーはショーダウンを待たずに即座に次ハンドへ移動する。
- MVPフェーズでは Bot難易度は1種類とする。

## 2. 残る未確定事項
- Board A / Board B それぞれのカード公開演出や表示レイアウトの細部
- Bot行動ロジックの具体的な実装仕様（難易度は1種類だが、行動戦略の詳細は設計フェーズで決定する必要あり）
- Pot A / Pot B の端数処理ルール（特に異なる勝者が複数いる場合の最終分配）
- フォールドプレイヤーの結果通知の見せ方（即時次ハンド移動時のUI表示）
