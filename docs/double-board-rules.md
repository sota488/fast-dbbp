# Double Board Bomb Pot Fast Fold ルール

## 1. 概要
- 2つの独立したボード（Board A / Board B）を使い、1つのハンドで2つの勝者判定を行うBomb Pot形式のFast Fold Omaha。
- プレイヤーは4枚のホールカードを受け取り、Board A と Board B の両方に同じホールカードで参加する。
- 1つのポットに賭けられたステークは、最終的に2つのボードに分割して配分する。

## 2. Board A / Board B
- Board A と Board B は完全に独立した5枚のコミュニティカードセット。
- それぞれのボードに対して独立にハンド評価を行い、各ボードごとに勝者を決定する。
- 同じプレイヤーが両方のボードで最強ハンドを持つ場合、そのプレイヤーが合計ポットをすべて獲得する。

## 3. ハンドフロー
1. プレイヤーに4枚のホールカードを配る。
2. Board A に5枚、Board B に5枚のコミュニティカードを配る。
3. 各プレイヤーはBoard A / Board Bそれぞれについて、ホールカード4枚とボード5枚を用いて最強のOmaha手を評価する。
4. 各ボードごとの勝者を決定し、ポットを2つのボードに分割して配分する。
5. その結果に基づき、各プレイヤーの獲得額を確定する。

## 4. Fast Fold とフォールドの振る舞い
- Fast Foldでは、プレイヤーがフォールドした場合、**ショーダウンを待たずに次のハンドへ移動できる**。
- フォールドしたプレイヤーは、そのハンドについてBoard A / Board Bの勝利対象から除外される。
- ただし、フォールド時点で既にコミットされたチップは、ハンドのポットに残り、残りのアクティブプレイヤーに配分される。
- UIやキューは、フォールドしたプレイヤーをショーダウン待ち状態にせず、速やかに次のハンドへ移動させる。

## 5. 勝者判定
- 各ボードごとに、参加プレイヤーの最強Omahaハンドを比較する。
- 1つのボードの最強手が複数名でタイの場合、そのボードの配分はタイ持ち間で均等分割される。
- Board A と Board B は別々に評価・分配される。
- 同一プレイヤーが両方のボードで勝利した場合、そのプレイヤーが総ポットを獲得する。
- ボードごとの勝者が異なる場合、各ボードの半分ずつを別々に配布する。

## 6. 重要な用語
- Pot Split: 総ポットをBoard AとBoard Bの2つに分割する動き。
- Quarter Pot: Board AまたはBoard Bそれぞれの半分をさらに複数の勝者で分割した際に生じる、総ポットに対する1/4の単位。
- Multi-way Split Pot: 1つのボードで2人以上のプレイヤーがタイになり、そのボードシェアを均等に分割すること。

## 7. 例
- 同一プレイヤーがBoard A / Board Bの両方を制覇 → そのプレイヤーが総ポット全額を獲得。
- Player AがBoard Aを勝利、Player BがBoard Bを勝利 → Aが総ポットの50%、Bが50%を獲得。
- Board AでAとBがタイ、Board BでCが勝利 → AとBがそれぞれ総ポットの25%ずつ、Cが総ポットの50%を獲得。

## 8. 追加ルール
- 6-max形式を前提とし、最大6人まで参加できる。
- 1ハンドあたりの参加者が少数でも、Board A / Board Bの評価は独立して行われる。
- フォールドしたプレイヤーのハンドは公開する必要はなく、単に勝利対象から除外される。
- すべてのカード情報と配分結果は、Board A / Board Bごとに別々に保持する。
