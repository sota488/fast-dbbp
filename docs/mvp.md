# MVP定義 - Fast Fold Omaha Practice

## 1. MVP ビジョン
Fast Fold Omaha練習サイトのMVP（Minimum Viable Product）は、**ブラウザで1人がBot5人と対戦し、複数ハンドを高速に切り替えながら遊べるOmahaゲーム**です。最短経路で「遊べる」状態を実現し、Fast Fold体験を最優先とします。

## 2. MVP判定基準
- 1人のプレイヤーが確実にゲームをプレイ可能
- 6-max（1人 + 5Bot）テーブルで常にゲームが進行
- フォールド後、次ハンドへ即座に移動可能（Fast Fold体験）
- ブラウザのみで動作（外部依存なし）
- 認証・DB・サーバー不要
- Play Money機能が機能している
- 基本的なUI操作（ボタンクリック）で全ゲーム進行可能

## 6. 開発フェーズ別計画

### Phase 1: Core Game Engine & Basic UI （2-3週間）
**目標**: 基本的に遊べる1テーブル

#### 実装項目
1. **ゲームロジック基盤**
   - カードデッキ管理（シャッフル、配布）
   - ハンド評価エンジン（60通り組み合わせ）
   - Pot Limit ベッティング計算
   - プリフロップ・フロップ・ターン・リバー進行

2. **シンプルなUI**
   - HTML/CSS基本レイアウト
   - カード表示（ASCII or 簡単な画像）
   - アクションボタン（Fold, Check, Call, Raise）
   - 現在のテーブル状態表示

3. **プレイヤー・ボット管理**
   - 人間プレイヤーの入力処理
   - ルールベースBot実装（簡単な戦略）
   - 各プレイヤーの残高管理

4. **ハンド完了フロー**
   - ハンド終了時のポット分配
   - 勝敗判定・結果表示
   - 残高更新

#### デリバリー
- `index.html` （ブラウザ入口）
- `game-engine.js` （ルール実装）
- `hand-evaluator.js` （ハンド評価）
- `bot-ai.js` （Bot行動）
- `ui.js` （画面操作）
- ブラウザで開くだけで遊べる状態

#### テスト重点
- ハンド評価テスト（100+ TC）
- ベッティング有効性
- ゲーム終了フロー
- 残高計算

---

### Phase 2: Fast Fold Queue & Session Management （1-2週間）
**目標**: Fast Fold体験を実現

#### 実装項目
1. **Fast Fold Queue実装**
   - PlayerPool 管理
   - FIFOキュー（waitingQueue）
   - テーブル再配置ロジック
   - 同一ハンド再参加禁止（participatedHandIds）

2. **マルチハンド進行**
   - 複数テーブル同時管理（ただし1人プレイヤーは1テーブルのみ）
   - ハンド完了→自動次テーブル割り当て
   - フォールド→即座な次テーブル移動

3. **セッション管理**
   - localStorage での状態保存
   - ブラウザリロード後の復帰機能
   - セッション初期化（新規ゲーム開始）

4. **UI強化**
   - Fast Fold切り替え時のビジュアル通知
   - ハンド履歴ログ（最近10ハンド）
   - 基本的な統計（ハンド数、勝率）

#### デリバリー
- `fast-fold-queue.js` （キュー管理）
- `session-manager.js` （セッション保存）
- 更新された `game-engine.js` （マルチハンド対応）
- 更新された `ui.js` （ハンド履歴、統計）

#### テスト重点
- 60通りハンド評価の正確性
- キュー管理（再配置、禁止）
- セッション復帰
- ハンド完了フロー

---

### Phase 3: Polish & Optimization （1週間）
**目標**: MVP完成・リリース準備

#### 実装項目

   - キーボード操作対応
   - アニメーション（フェード、スライド）

3. **ゲーム設定UI**
   - ブラインド設定
   - 初期残高設定

4. **デバッグ・最適化**
   - コードクリーンアップ

## 7. MVP スコープ外（明示的に除外）

| 項目 | 理由 |
|------|------|
| サーバー / API | ブラウザのみで動作するため不要 |
| データベース | localStorage で十分 |
| ユーザー認証 | シングルクライアント、アカウント管理不要 |
| マルチプレイヤーネットワーク | 1人プレイのため不要 |
| リアルマネー / 課金 | Play Money のみ |
| モバイル対応 | デスクトップ・タブレットのみ |
| 高度なBot | ルールベースの簡単な戦略のみ |
| 統計分析ダッシュボード | 基本統計のみ |
| ビデオレッスン | 後の機能 |
| トーナメント管理 | キャッシュゲームのみ |

## まとめ

FastFold Omaha MobileのMVPは、**ブラウザ上で1人がBot5人と高速にハンドを切り替えながら遊べるOmahaゲーム**です。

- **スコープを絞る**: 認証・DB・サーバーなし
- **Fast Fold体験を優先**: 高速な次ハンド割り当て
- **シンプルなUI**: クリック操作のみ
- **3フェーズ開発**: 6週間で完成

MVPを完成させることで、ユーザーフィードバックを得て、次のフェーズへ展開します。
